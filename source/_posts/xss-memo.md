## 过滤函数 ##

主要是转义and、小于号、大于号和两个引号。

```
function htmlEnco(s)
{
    return s.replace(/</g, "&lt;")
            .replace(/>/g, "&gt;")
            .replace(/&/g, "&amp;")
            .replace(/"/g, "&quot;")
            .replace(/'/g, "&#39;");
}
```
